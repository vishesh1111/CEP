# Quick Start: Camera QR Code Scanner

## ✅ Feature Complete & Ready to Use

The admin check-in page now supports **direct camera scanning** of participant QR codes!

---

## 🚀 Quick Test (2 Minutes)

### Step 1: Open Check-in Page
1. Navigate to: **Admin Panel** → **Check-in**
2. Or directly visit: `http://localhost:3000/admin/check-in`

### Step 2: Start Camera
1. Click **"Start Camera"** button
2. Allow camera access when browser asks
3. Camera feed appears in the scanner box

### Step 3: Scan a QR Code
1. Open a participant's event registration on another device
2. Show their QR code to your camera
3. ✨ **Automatic check-in happens!**
4. Success message shows participant name
5. Details appear in "Last Check-in" card

### Step 4: Stop Camera (Optional)
- Click **"Stop Camera"** when done
- Camera automatically stops when you leave the page

## 📱 Three Ways to Check In

### Method 1: Camera Scan (Recommended) ⭐
```
Click "Start Camera" → Point at QR code → Done!
```
- **Fastest method**
- Best for continuous check-ins
- Works with phone/tablet cameras

### Method 2: Image Upload
```
Drag QR code screenshot → Drop on upload zone → Done!
```
- Good for remote check-ins
- Works with photos/screenshots
- No camera needed

### Method 3: Manual Entry
```
Type QR code → Press Enter → Done!
```
- Backup method
- Good for troubleshooting
- Works offline (with connection for validation)

## 💡 Tips for Best Results

### During Scanning
- 📏 **Distance**: 15-30cm from camera to QR code
- 💡 **Lighting**: Ensure good lighting on QR code
- 🎯 **Angle**: Face QR code directly at camera
- ⏱️ **Steady**: Hold still for 1-2 seconds

### For Events
- 🖥️ **Use tablet/laptop** at entrance
- 🔋 **Keep device plugged in**
- 📶 **Ensure stable WiFi**
- 🔄 **Have backup device ready**
- 👥 **Brief staff beforehand**

## 🐛 Quick Troubleshooting

### Camera Won't Start?
1. ✅ Check browser permissions (click camera icon in address bar)
2. ✅ Close other apps using camera (Zoom, Teams, etc.)
3. ✅ Reload the page
4. ✅ Try different browser

### QR Code Not Scanning?
1. ✅ Improve lighting
2. ✅ Adjust distance (15-30cm)
3. ✅ Hold steady for 2 seconds
4. ✅ Increase screen brightness (if scanning from phone)
5. ✅ Try manual entry as backup

### "Already Checked In" Error?
- This participant was already checked in
- Check "Recent Check-ins" list to verify
- This is working as expected (prevents duplicates)

## 💬 Common Questions

**Q: Does this work on mobile devices?**  
A: Yes! Works great on phones and tablets with Chrome/Safari.

**Q: Can I use my laptop webcam?**  
A: Absolutely! Works with any camera (laptop, external, phone).

**Q: What if the camera is blurry?**  
A: Wait 1-2 seconds for auto-focus, or adjust distance.

**Q: Can multiple admins scan at once?**  
A: Yes! Each admin can scan on their own device simultaneously.

**Q: Does this work offline?**  
A: Camera works offline, but needs internet to validate/record check-ins.

**Q: What happens if I scan the same code twice?**  
A: Second scan will show "already checked in" error. Each QR code works only once.
